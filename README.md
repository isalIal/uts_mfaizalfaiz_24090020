## Deskripsi Singkat Project

Project ini adalah implementasi front-end dari desain antarmuka web yang mencakup tiga halaman utama menggunakan HTML, CSS dasar, dan JavaScript untuk manipulasi DOM dan pengelolaan data dummy.

- **1. Halaman Login (`index.html`):**
  - Fungsi: Form login simulasi.
  - Fitur: Validasi input **Email** dan **Password (NIM)**. Jika sukses, akan **redirect** ke `dashboard.html`.
- **2. Halaman Dashboard (`dashboard.html`):**
  - Fungsi: Menampilkan ringkasan data produk (Total Produk, Penjualan, Revenue).
  - Fitur: Data diambil dari array dummy di `script.js`. Menggunakan **Grid CSS** untuk tata letak card.
- **3. Halaman List Data Produk (`products.html`):**
  - Fungsi: Menampilkan daftar produk dalam format tabel.
  - Fitur: Data diambil dari array dummy. Menggunakan JavaScript (**forEach**) untuk merender tabel. Terdapat tombol **Edit** (menampilkan `alert`) dan **Delete** (dengan konfirmasi `confirm()` dan menghapus baris dari DOM).
